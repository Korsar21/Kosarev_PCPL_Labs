#  Telegram Habit Tracker Bot

Telegram-бот для формирования привычек, построенный на архитектуре **конечного автомата (Finite State Machine)**. Помогает пользователю выработать полезную привычку через ежедневные напоминания и контроль выполнения.

##  Описание

Проект демонстрирует реализацию конечного автомата на Python в контексте Telegram-бота. Бот проводит пользователя через челлендж по формированию привычки: от выбора привычки до успешного завершения или провала.

##  Конечный автомат (FSM)

### Описание состояний

| Состояние | Описание | Переход |
|-----------|----------|---------|
| `SAVE_HABIT` | Ожидание ввода названия привычки | → `SAVE_PERIOD` |
| `SAVE_PERIOD` | Ожидание ввода количества дней | → `NOT_STARTED` |
| `NOT_STARTED` | Подтверждение параметров челленджа | → `IN_PROGRESS` |
| `IN_PROGRESS` | Активное выполнение челленджа | → `FINISHED` или → `SAVE_HABIT` (провал) |
| `FINISHED` | Челлендж успешно завершён | → `SAVE_HABIT` |

##  Реализованный функционал

###  Настройка челленджа
- Ввод названия привычки (любой текст)
- Указание продолжительности в днях
- Валидация ввода (проверка на число)
- Подтверждение параметров перед стартом

###  Система напоминаний
- **Ежедневные напоминания** о выполнении привычки
- **Гибкое откладывание** — кнопка "Позже" с повторным напоминанием через час
- **Автоматический переход** к следующему дню после выполнения

###  Система успеха/провала
- **Успех:** выполнение задания каждый день в течение всего периода
- **Провал:** слишком много откладываний за день (накопленная задержка превышает 24 часа)
- **Счётчик дней:** отслеживание прогресса

###  Кнопочный интерфейс
- `Погнали` — начать челлендж
- `Выполнил` — отметить выполнение задания
- `Позже` — отложить напоминание на час
- `Ура!` — подтвердить успешное завершение
- `Эхх` — подтвердить провал

##  Технологии

- **Python 3**
- **pyTelegramBotAPI (telebot)** — взаимодействие с Telegram Bot API
- **Enum** — реализация состояний автомата
- **Конечный автомат (FSM)** — архитектурный паттерн управления состояниями
